# http4k Pro


**http4k Pro** is a set of commercially-licensed modules that extend http4k with production-grade
capabilities - enhanced security, observability, AI tooling and developer productivity - for teams running
http4k in regulated, high-volume environments.

Every Pro module is built on the same functional foundation as the open-source core and held to the same
testing standards. They add enterprise-focused features, not a different way of working - so adopting one is
just another dependency, with the same in-memory testability you already rely on. Browse the modules in the
sidebar, or see [licensing and pricing on the Pro page](/pro/).

Pro modules are free for personal use, non-profits and qualifying small businesses. They can be licensed
standalone under the [http4k Commercial License](/commercial-license/), or come bundled as part of the
[http4k Enterprise Edition](/enterprise/).
